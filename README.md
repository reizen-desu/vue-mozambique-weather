## Description

This is an app to display weather information, mainly for the [provinces of the Mozambique](https://en.wikipedia.org/wiki/Provinces_of_Mozambique) country. You can select the province you want to see the weather information.

Implemented using the [Vue](https://vuejs.org/) framework and the [Open Weather API](https://openweathermap.org/) to fetch the weather information.

### Concepts applied

- Single File Components
- Fetching data from API
- Passing data between child and parent Components
- Deployment on GitHub

## Project setup
